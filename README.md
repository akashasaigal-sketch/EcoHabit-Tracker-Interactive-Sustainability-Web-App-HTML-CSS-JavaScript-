🌱 EcoHabit Tracker

EcoHabit is a simple and interactive web application built with HTML, CSS, and JavaScript that encourages users to adopt eco-friendly habits. Users can perform positive environmental actions to earn energy points and watch their virtual tree grow as they progress.

🚀 Features
Interactive tree growth system
Eco-friendly action tracking
Energy points and level progression
Positive and negative environmental choices
Clean and modern user interface
Real-time updates using JavaScript
Engaging gamification experience
🛠 Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
SVG Graphics
📖 Project Overview

EcoHabit helps users visualize the impact of their daily environmental choices. Actions such as recycling, saving water, biking, and planting trees increase energy points, while harmful activities reduce them. As users earn more points, their virtual tree grows, creating a fun and motivating sustainability experience.
